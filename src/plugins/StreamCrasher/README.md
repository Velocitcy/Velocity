# Stream Crasher

Stream crasher is pretty self explanatory, just crashes/freezes your stream and nobody can see

This comes with the AutoJoinCall plugin as the stream to crash

You can turn it on/off it with the button or with a **F7** key press. shown in the image. You can also change the button position and have a chevron button for quick access
